About Resources

Resources is a shared collection of learning materials for students and researchers in manuscript studies. Each contributor has a personal folder in the MANO GitHub repository where they can upload their PDFs, slides, and other resources. The materials are freely available for students and published under the Creative Commons Attribution - NonCommercial (CC BY-NC) license. Anyone can reuse them for educational purposes, but must always credit the author and cannot use them commercially.

How can I contribute?

All learning materials are stored in the MANO GitHub repository. To contribute, you need a free GitHub account. Once you have it:

  1. Create your personal folder in the MANO Resources repository
  2. Upload your materials (PDFs, slides, etc.) into your folder
  3. Your files will be automatically displayed on this page

For a step-by-step guide with detailed instructions, click the “Contribute to the Resources” button in this page. It will redirect you to the GitHub repository, where you will find the full documentation and the upload process.

Every contribution helps expand this shared learning resource, and participation in enriching the MANO Resources collection is greatly appreciated.
