Metadata Editor

About Metadata Editor

The Metadata Editor enables the structured creation and management of manuscript metadata in full compliance with the TEI P5 manuscript description guidelines. A single session may include the description of one or multiple manuscripts. Each manuscript can be exported in TEI-XML or JSON formats, while all manuscripts described during the session may also be downloaded collectively in JSON.

Existing metadata files, provided in JSON or TEI-XML, may be reloaded into the editor. When the file is compliant with the metadata structre implemented in MANO, the information is automatically restored into the form, allowing previously interrupted work to be resumed seamlessly without the need for manual re-entry.

JSON metadata files can be uploaded to the MANO GitHub repository. Once accepted, these records will enrich the publicly accessible MANO Metadata Collection, enriching the shared digital resource for manuscript research.
